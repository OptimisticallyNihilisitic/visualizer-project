# Data Visualizer App

A PowerBI-like data visualization web app built with React, TypeScript, and Yarn.  
It allows users to upload Excel/CSV files, select columns, and visualize data in
interactive tables.

## Getting Started

### Prerequisites

- Node.js >= 18
- Yarn

### Installation

```bash
git clone https://github.com/OptimisticallyNihilisitic/visualizer-project.git
cd visualizer-project
yarn install
```
### Run the app
```bash
yarn dev
```
The app will be available at:
http://localhost:5173

### Usage
1. Upload a CSV or Excel file
2. View extracted columns in the Data Panel
3. Select columns in the Visualization Panel
4. View the data rendered in a table
5. Change category to update results dynamically
